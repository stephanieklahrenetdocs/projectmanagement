⚙️ Calendar Sync Settings

Orbit uses the Anthropic API to pull live calendar events from your Microsoft 365 account. Your key is stored only in this browser — never sent to any server except Anthropic.

Get your key at console.anthropic.com
How it works: Sync Meetings fetches calendar events + Teams transcript URLs via M365 MCP. When you expand a meeting recap, the transcript is fetched and summarized by Claude automatically. Meeting lists are cached 24h; individual recaps are cached indefinitely until regenerated.
