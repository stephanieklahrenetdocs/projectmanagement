⚙️ Integration Settings

Orbit uses Asana to pull live meeting data for each client project. Your token is stored only in this browser — never sent to any server except Asana.

Get your token at app.asana.com/0/my-apps
How it works: "Refresh from Asana" searches for the client's Asana project, then loads tasks with a month-prefix name (e.g. "MAR 30 - Kickoff"). Attendees and recaps are read from task notes. Results cache for 1 hour; click Refresh to force a reload. Without a PAT, pre-built static recaps are shown instead.
